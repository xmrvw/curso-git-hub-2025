ls -> lista arquivos
cd -> move entre diretórios
pwd -> mostra diretórios
cat -> exibe conteúdo do arquivo
rm -> remove arquivo ou diretório
rm - rf -> remove de forma recursiva e forçada todos diretórios

### FLUXO DE TRABALHO
1. criar o diretório do projeto
2. iniciar o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3.
